Position

The position property let's us control the way an element is positioned inside our HTML document.

Values

The position property can have 5 possible values:

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky

We say that an element is positioned when it has the position property with any value other than static: in other words, any value except the default one.

Controlling the position

Positioned elements (those with a position value other than static) enable the use of 4 properties that are otherwise ineffective:

  • top
  • right
  • bottom
  • left

Using any combination of those properties, we can change the position of an element inside the document.

Static elements

Every evelements by default is static. Static elements simply follow the flow of the document.

Relative positioned elements

The initial position of the element is the same as for the static one. The difference is that with relative elements we can now use the top, right, bottom, left properties to move them by a certain offset relative to themselves.

For example:

    
.relative {
    position: relative;
    left: 50px;
}
